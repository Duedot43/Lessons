You can again use the print() function to print a variable on the screen
Code example:
     #setting up the variable
     var = 3
     #printing the variable
     print(var)
     Output:
          3
     #You can also print strings in a variable
     var = "Hi!"
     #Print the variable
     print(var)
     Output:
          Hi!
You can also print text around a string with print()
Code example:
     #setting up the variable
     var = "Hi!"
     #printing the variable
     print(var, "friend")
     Output:
          Hi! friend
