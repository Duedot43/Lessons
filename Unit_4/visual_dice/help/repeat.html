How do i repeat a section of code until a condition is met?
you can use a while loop.
A while loop is where it repeats a section of code until a condition is met.
Code example:
     #set the game variable
     game = 1
     #setup the while loop
     while game == 1:
          # == is just basicly = != is does not =
          #get the user input and put it in the input variable
          input = input("Play again?\n(1) Yes\n(2) No\n")
          if input == 1:
               continue
          elif input == 2:
               game = 0
