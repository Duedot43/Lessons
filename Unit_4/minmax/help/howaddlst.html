How do i add things to a list in python?:
     
     You can anything you can normally pyt in a variable in a list including variables!
     
     You can append something with list.append()
     Example:
     
          #set the variable I want to add to the list
          var = 3
          #set up the list
          myList = [0, 1, 2]
          #append the variable to the list
          myList.append(var)
          print(myList)
          Output:
               [0, 1, 2, 3]

     You can also add strings to lists:
          #set the variable I want to add to the list
          var = "HTML"
          #set up the list
          myList = ['hi', 'from']
          #append the variable to the list
          myList.append(var)
          print(myList)
          Output:
               ['hi', 'from', 'HTML']

     You can also add variables to specified parts of a list
     Example:
          #set the variable I want to add to the list
          var = "HTML"
          #set up the list
          myList = ['hi', 'from', 'what']
                          #0     1         2
          #append the variable to the list
          myList[2] = "HTML"
          print(myList)
          Output:
               ['hi', 'from', 'HTML']
