What is a list in python?:
     
     A python list is a list of objects in a single variable.
     
     A integer list looks like this
     myList = [1, 2, 3]
     
     A float list looks like this
     myList = ['1.3', '4.26', '5.23']
     
     A string list looks like this
     myList = ['Duedot43', 'Is', 'Not', 'Good', 'At', 'HTML']
