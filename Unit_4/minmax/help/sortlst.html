How to sort a list with the sort() function?
The list.sort() funtion has many uses but for this example we wont be using any of these.
The list.sort() function with no command arguments (list.sort(arguments)) sorts the list by least to greatest number.
Example:
     #set up our list
     list = [2, 5, 2, 4, 5, 2, 7]
     #sort the list
     list.sort()
     #print the list
     print(list)
     Output:
          [2, 2, 2, 4, 5, 5, 7]
