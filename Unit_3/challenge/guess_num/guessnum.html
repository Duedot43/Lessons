Your task is to create a 2-player guess the number. It should have the following:
-Welcome Screen
-Ask for Players' names
-Rules screen
-Randomly generate a number between 1 - 100
-Have both players make a guess
-Report what the random number was and who was closest
-Give that player a point
-Play to 5 points
-Ask if you want to play again
 
Your program should include the following:
-Data Validation
-Randomize turn (The same player shouldn't always be going 1st)
-Utilize while loops
 
-When you get the basic program done, see what enhancements you can add (Maybe if you're exactly on the number you get 5 points, if you are exactly half the number you get 2 points, maybe there is power-ups that can happen or can be purchased? Maybe give hints about the generated number (i.e. it's an even number, it's divisible by 3, etc.)
