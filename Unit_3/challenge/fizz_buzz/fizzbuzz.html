FizzBuzz is a classic coding interview question. It was invented by Imran Ghory, and popularized by Jeff Atwood. The text of the programming assignment is as follows

Write a program that prints all numbers from 1 to 100. BUT, for multiples of three, print “Fizz” instead of the number and, for multiples of five, print “Buzz” instead of the number. For numbers that are multiples of both three and five print “FizzBuzz".

You may think the FizzBuzz problem is too easy of an interview question – and it is blindingly, intentionally easy – a commenter to Imran's post notes its efficacy: "I'd hate interviewers to dismiss [the FizzBuzz] test as being too easy - in my experience, it is genuinely astonishing how many candidates are incapable of the simplest programming tasks."

Problem Statement

Ask the user what number they want to print to:

Use data validation to make sure they are typing in a positive integer

print out all the numbers up to the one that the users inputted

Replace all the numbers divisible by 3 with the word "Fizz"

Replace all the numbers divisible by 5 with the word "Buzz"

If the number is divisible by both, concantanate together "FizzBuzz"
