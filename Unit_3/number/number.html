Your challenge is to create a guess the number game:
- The program will randomly generate a number between 1 - 20. The user will have 4 guesses to try and get it right. If they do get it right, they win. If they run out of guesses, they lose.
 
The program should do the following:
 
 - Explain the rules of the game (The computer will select a ranom number between 1 and 20, you have 4 guesses to get it right)
- Ask for the user's name
- Have the program randomly choose a number but keep it hidden from the user
- Ask for the player's guess
- Use data validation to make sure guess is whithin the range of 1 and 20
- Compare the player's guess to the secret number and give feedback (your guess was too high, too low)
- Tell the user how many guesses they have left
- If the player still has guesses left, let them make another guess
- Game ends if the user gets the number right or runs out of guesses
